Flights Search

Submit a request to http://localhost:8081/weather

Request Input parameters are:

  • name
  • lastName
  • datetime
  • city
  • country

Sample Request:

  • { "OpenWeatherRequest":[{ "name":"John", "lastName":"Smith", "datetime":"2020-01-01 10:00:01", "city":"Melbourne", "country":"Australia" }] }
